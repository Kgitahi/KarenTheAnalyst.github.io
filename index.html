Stock Portfolio Optimization in python

The Python Portfolio Optimization project leverages historical data to optimize a diverse set of assets, including SPY, BND, GLD, QQQ, and VTI. By calculating adjusted close prices, lognormal returns, and a covariance matrix, the project provides insights into asset volatility and diversification. Using metrics like portfolio variance, expected return, and the Sharpe ratio, it helps investors make informed decisions to maximize returns and minimize risk.

Fair's Company linear Programming Maximization of Profits Problem

This project demonstrates a linear optimization model for maximizing profit in furniture production using Python, Pyomo, and GLPK. The model optimizes the number of tables and chairs to produce, subject to constraints on carpentry and painting hours, and production limits. The solver finds the optimal solution, yielding a maximum profit of 4150 units with 200 tables and 450 chairs produced.

Capital Budget Distribution City Of NYC - Machine Learning Project

This project involves analyzing and cleaning New York City capital budget data using Python and various data preprocessing techniques. The dataset, sourced from NYC Open Data, includes budget allocations for different fiscal years and project types. The project converts date columns to datetime format, handles missing values, and scales numerical features. Categorical data is one-hot encoded for analysis. Visualization techniques like heatmaps and PCA (Principal Component Analysis) are applied to understand correlations and the variance explained by principal components.

SQL Database Creation - Conceptual and logical Modelling

This project involves developing a robust and efficient SQL database for a marketing agency, Creators.Inc, to manage relationships with influencers, brands, and employees.
